# T Flipflop

A T flip-flop, also known as a Toggle flip-flop, is a fundamental  element in counters. It Toggle the state when ever the input T is high (at edge of the clock), otherwise it holds is value.

| Reset | $T$ | $Q$            |
| ----- | --- | -------------- |
| 1     | $X$ | $0$            |
| 0     | 0   | $Q$            |
| 0     | 1   | $\overline{Q}$ |
